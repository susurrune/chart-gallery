电脑还能
这么用？

你每天面对这台机器至少八小时，却只见过它一面。在 GUI 之下、在默认设置之外、在「正常用法」的边缘——藏着一个等待被重新发现的工具箱。

探索
01 / 认知边界

你只用了它
十分之一的灵魂

我们用电脑的方式，固化得太早了。双击图标、打开软件、开始工作——这套流程如此自然，以至于没人停下来问一句：这个功能还能用来做什么？

快捷键不只是省时间，它们改变的是你和机器之间的对话方式。脚本不只是偷懒，它们是你意志的延伸。系统深处那些从未被点开的设置面板，每一个都可能重新定义你与工具的关系。

真正的「会用电脑」，不是学会某个软件的操作手册，而是建立起一种本能：在任何界面中，都能嗅到那些未被言明的可能性。

02 / 七种武器

键盘之外的键盘

01

键盘即乐器

把键盘当作一件真正的乐器来演奏，而不只是打字输入的工具。

Live Coding（实时编程音乐） 是一种正在兴起的表演艺术形式——程序员用代码实时生成电子音乐，每一行新指令都即时改变声场。Sonic Pi 是入门首选：它以 Ruby 语法封装了合成器引擎，你只需写 play :C4 就能听见钢琴声，写 3 行 loop 就能织出一段循环节奏。FoxDot 更进一步，它让 Python 代码与 SuperCollider 合成器引擎对话，支持用算法定义旋律、节奏、和声，甚至将音乐参数映射到视觉粒子系统上。

模块化合成器 也从硬件走进了软件。VCV Rack 免费开源，它模拟了 Eurorack 硬件模块合成的完整信号链路：振荡器 → 滤波器 → 包络 → VCA，你可以随意跳线、组合、调制。Orca 则另辟蹊径——它是一个二维网格编程语言，音符在格子间像生命游戏一样扩散、碰撞、触发音色。花一个下午搭建一个模块链，你会发现声音设计本质上和写代码是同一件事：定义输入，控制流，观察输出。

创作
02

快捷键里的宇宙

鼠标是人类输入效率最低的发明之一，这个论断有扎实的认知科学依据：从手离开键盘到握住鼠标、定位、点击、再回到键盘，平均需要 0.8-1.2 秒。一天 200 次这样的切换，浪费的时间接近 20 分钟——一年就是 80 多个小时。

真正的高手把操作系统调教成一把乐器，双手不离开键盘，思维不被打断。但快捷键的真正威力不在于单个组合，而在于组合的组合：比如在 macOS 上，⌘⇧4 截图后按住 Control 可直接存入剪贴板而非生成文件；⌘Tab 切换应用后立即按 ⌘` 可切换同一应用的窗口；在 Finder 中 ⌘⇧G 直达路径后 ⌘↑ 快速回到上级目录。

窗口管理 是效率提升的最大捷径。Rectangle（macOS）或 PowerToys FancyZones（Windows）能把屏幕分割成自定义区域，窗口贴边即可自动吸附。学会这十组快捷键，你的操作速度至少翻倍。

macOS 必知: ⌘⇧4 精确截图 · ⌥⌘Space 聚焦搜索 · ⌘⇧G 跳转到路径 · ⌘⇧~ 同应用窗口切换
Windows 必知: ⇧S 截图 · V 云剪贴板历史 · ⇧←→ 窗口分屏 · D 显示桌面
效率
03

终端里的画布

你那个黑底绿字的终端，其实是一间暗房。当 GUI 成为负担，最原始的界面反而给你最完整的控制权——这也是为什么许多艺术家回归命令行。

图像转 ASCII 看似复古，实则有深刻的美学追求。chafa 和 jp2a 能把你的一张照片实时转为字符画，但差别巨大：chafa 支持 24 位真彩色、多种 dithering 算法，甚至能用 Unicode 半块字符实现接近原图一半分辨率的渲染效果。配合 --symbols 参数选择字符集（block/shade/braille），一张照片能变成几十种截然不同的艺术风格。

终端本身就是画板。 Pipes.sh 让彩色的管道在终端蜿蜒生长，Cmatrix 模拟《黑客帝国》数字雨——但别只把它们当屏保。更激进的玩法是用 tput 和 ANSI 转义码直接在终端绘制矢量图形：\e[48;2;R;G;Bm 可以为任意字符设置 24 位背景色，组合起来就是一块像素画布。配合 Shell 脚本循环，你可以做终端动画、实时数据可视化、甚至简单的物理模拟。

04

旧机器的第二人生

一台五年前的笔记本，真的就「不能用」了吗？问题的根源通常不是硬件，而是臃肿的最新版操作系统和应用。换上一个轻量级操作系统，它能做的事远超你想象。

家庭服务器 是最简单的起步：安装 Ubuntu Server（最小化只需 256MB 内存），装上 CasaOS 或 DietPi 作为管理面板，瞬间拥有 NAS 文件共享、Jellyfin 媒体流、qBittorrent 下载机、AdGuard Home 全网去广告 —— 一台老旧笔记本就能覆盖整个家庭的基础设施需求。

再激进一点： Pi-hole 做 DNS 级广告拦截，配置好后全家所有设备的广告在到达浏览器之前就被丢弃，连 App 内的开屏广告都能被过滤。Home Assistant 做智能家居中枢，只要旧电脑有一个 USB 口插上 Zigbee 网关，就能统一接管灯泡、传感器、窗帘、空调。旧电脑不是垃圾，它只是还没找到自己的角色。

05

自动化你的日常

一个简单的判断标准：每天重复三次以上的操作，就值得被自动化。这不仅省时间，更重要的是消除重复带来的认知疲劳——每次手动操作都是一次注意力中断。

文件自动化： Hazel（macOS）是文件整理的神器：设定规则「下载文件夹中 PDF 文件超过 24 小时 → 移动到 Documents/Papers」后，它就在后台静默执行。AutoHotkey（Windows）能做到更多——写 5 行脚本，把一个快捷键映射为「打开浏览器、输入搜索词、截取结果、存入 Obsidian」的完整流程。

工作流自动化： n8n 是自托管的 Zapier 替代品。拖拽节点就能搭建：收到带附件的邮件 → 提取内容 → 调用 AI 总结 → 写入 Notion 数据库 → 推送 Slack 通知的完整流水线。更进一步，Keyboard Maestro 可以录制你的操作序列并回放，甚至能在条件判断中做出「如果窗口标题包含 X，则执行 Y」这样的智能决策。把电脑从「被操作的对象」变成「替你操作的主体」，这才是它本该做的事。

解放
06

浏览器即操作系统

浏览器早已不只是浏览工具。在 WebAssembly 和 PWA 技术的支撑下，浏览器正逐步模糊「本地应用」和「网页应用」的边界。

WebAssembly 带来了什么？ C/C++/Rust 代码可以编译成 WASM 在浏览器中以接近原生的速度运行。Figma 的引擎用 WASM 运行 C++ 的矢量渲染核心，Photopea 用 WASM 加载了完整的 PSD 解析库——这就是为什么浏览器里的设计工具能和本地 Photoshop 掰手腕的原因。对于普通用户来说，这意味着你不再需要为了一个功能安装几百 MB 的软件。

PWA（渐进式 Web 应用） 是可安装、可离线运行、可接收推送通知的网页。一个 PWA 图标在你的 Dock/任务栏上看起来和原生应用毫无区别——但它背后只托着一个 URL。当你用 vscode.dev 写代码，用 Excalidraw 画白板，用 tldraw 做线框图，用 Photopea 修图，用 Notion 写文档，你的下一台电脑真的可以只是一个浏览器标签页。

07

用数据写诗

编程创作的核心从来不是技术，而是表达。代码是 21 世纪的鹅毛笔——你的表达欲望有多强，代码被赋予的使命就有多重。

数据可视化是最直接的跨界语言。 D3.js 不只是图表库，它让你把 GDP 数据变成山川河流——将数值映射到高度、颜色、透明度、甚至笔画粗细。改变一个映射函数，同一个数据集就能讲述截然不同的故事。Observable 笔记本让数据探索变成叙事：在交互式表格旁嵌入文本，在图表上方写批注，读者可以拖动滑块改变参数、实时观察图表变化——这不是报告，这是活的文档。

三维和视觉创作 也早已不依赖大型软件。Three.js 在浏览器中搭建可进入的 3D 空间，p5.js 让创意编程的入门门槛降到几乎为零——写 10 行代码就能生成一幅动态艺术作品。ShaderToy 上用 GLSL 写的像素着色器，每一帧都是数学公式在 GPU 上实时计算的独一无二的图像。从数据到诗意，从逻辑到审美——代码正在模糊「工程师」和「艺术家」的界限。

03 / 四种心法

工具的价值
取决于你如何
看待它

电脑不是冰箱——它不只是被动保存内容的容器。它是一种媒介、一座乐器、一间暗房、一个可以无限重塑的宇宙。但大多数人终其一生只把它当作一个「装有软件的盒子」来使用。

认知科学家 Don Norman 在《设计心理学》中提出了「可供性」（affordance）的概念：一个物体能做什么，不仅取决于它的物理属性，更取决于使用者感知到的可能性。你把电脑看作打字机，它就只是打字机。你把电脑看作乐器、画布、出版厂牌、自动化工厂——它就真的是。

关键在于你有没有停下来问：
等等，这个还能怎么用？

01

观察

一个功能不只是它看起来的样子。右键菜单里那些你从未点开的选项、状态栏上不起眼的图标、设置面板中默认被折叠的高级选项——下次打开任何一个软件时，花 10 分钟系统性地翻遍每一个菜单。你极大概率会发现一些从未见过的功能，而它们恰好能解决你昨天还靠手动操作的某个难题。

02

拆解

把工具当乐高。从来没有一个功能是孤立存在的。这个软件的通知机制 + 那个脚本的触发条件 + 另一个服务的 API = 一条全新的工作流。当你在一个 App 中看到「支持 URL Scheme」或「有 CLI 接口」时，不要跳过——那是它愿意被你的意志组合的信号。

03

重构

不要接受默认。你的工作流应该由你设计，而不是软件供应商的产品经理决定。快捷键可以自定义、工具栏可以重排、自动保存路径可以修改、邮件规则可以自己写。每当你觉得「这软件用起来总有不顺手的地方」，那不是你的问题——是你还没有夺回对工具的控制权。

04

分享

知识因分享而增值。你花半小时发现的技巧，可能替另一个人省下半天。每次分享，都在重新定义「电脑还能这么用」这个问题的边界。写一篇博客、发一条推文、给同事演示一次——教是最好的学，也同时让这个社区的知识密度再涨一点。

每台电脑里都藏着一架钢琴、一块画布、一座图书馆和一间实验室。
大多数人终其一生，只用过它的文件管理器和浏览器。
